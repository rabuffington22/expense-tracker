# Agent Selection

This project should use the parent Projects Project agent-selection policy when available. If this project is running standalone, treat this file as the local policy surface and update it when model capability, tool availability, pricing, reliability, or Ryan workflow preferences materially change.

Skills and child project files should not hard-code durable rankings for which named AI agent is best. They may describe the role needed, required safeguards, and local availability, then check this policy before recommending a named agent.
